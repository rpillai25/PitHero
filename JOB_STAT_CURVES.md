# Primary Job Stat Growth Curves

This document details the stat progression for all 6 primary jobs in the game. All values represent job contribution only (excluding hero base stats and equipment).

## Stat Calculation Formulas

- **Stat at Level**: `BaseBonus + (GrowthPerLevel × (Level - 1))`
- **HP**: `25 + (Vitality × 5)`
- **MP**: `10 + (Magic × 3)`
- **Caps**: HP ≤ 9999, MP ≤ 999, Stats ≤ 99

## Implementation Details

### Technical Notes

1. **Float-Based Growth**: StatBlock internally stores float values to enable fractional growth rates (e.g., 0.612 per level). Values are rounded when accessed via properties.

2. **Linear Progression**: All stats grow linearly from level 1 to 99. The formula ensures predictable and balanced progression.

3. **Stat Caps**: All calculations respect maximum caps (HP ≤ 9999, MP ≤ 999, individual stats ≤ 99).

4. **Job Contribution Only**: Values shown represent job contribution only. Actual in-game stats also include:
   - Hero base stats (racial/innate stats)
   - Equipment bonuses
   - Buff effects

5. **Testing**: All jobs have been validated to reach their target stats at level 99 with 0% error and stay within caps at all levels.

## Implementation Notes

### Secondary Job Design Philosophy

1. **Hybrid Roles**: Secondary jobs combine the strengths of their parent primary jobs, creating unique playstyles.

2. **Power Scaling**: Secondary jobs have 15-25% higher stats than primaries, making them stronger but requiring more investment to unlock.

3. **Specialization**: Each secondary job excels in specific areas based on parent traits:
   - Physical + Physical = Elite Physical (Samurai, Ninja, DragonFist)
   - Physical + Magic = Hybrid Combat (DivineFist, WarMage, Paladin)
   - Physical + Speed = High Mobility DPS (KiShot, Stalker, Marksman)
   - Magic + Magic = Supreme Caster (Wizard, ArcaneArcher)
   - Magic + Speed = Evasive Magic (Spellcloak, Shadowmender)

4. **Balance**: Despite higher stats, secondary jobs respect the same stat caps (99) and HP/MP caps as primaries.
